# 연산자 오버로딩

## 사용 방법

1. Visual Studio에서 `Chapter37-OperatorOverloading.slnx`를 엽니다
2. 솔루션 탐색기에서 프로젝트를 선택하고 `README.md`를 읽습니다
3. `Program.cs`에 코드를 작성합니다
4. 해당 프로젝트를 시작 프로젝트로 설정하고 실행합니다 (`Ctrl+F5`)

## 코딩 연습

`CodingPractice` 프로젝트의 README를 따라 교안 예제 코드를 직접 작성하고 실행합니다.

| 프로젝트 | 내용 |
|----------|------|
| CodingPractice-01 | 교안 코드 따라치기 - 단항/이항/비교 연산자, 복합 대입, 암시적 변환 (5개 과제) |
| CodingPractice-02 | 교안 코드 따라치기 - 명시적 변환, 실전 예제 (2개 과제) |

## 연습문제

각 프로젝트의 README에 있는 문제를 읽고 코드를 작성합니다.

| 프로젝트 | 문제 |
|----------|------|
| GameCurrency | 게임 화폐 시스템 |
| GameTime | 게임 시간 시스템 |
